# niconicoProcessor

## RISC-V Microprocessor

![RV32I](RV32I.png)

### Features
- **Baseline** RISC-V ISA (RV32I).
- In-Order Execution.
- 32bit Word Datapath.
- ALU, Ld/St Unit, Branch Unit, CSR.
- Register Rename and Reorder Buffer are supported.
- Exception and Interrupt are **not** supported.
